## Project Description
This project is a simple personal About Me webpage. It introduces who I am, shares my hobbies and interests, includes a profile image placeholder, and provides a contact email link.

## Technologies Used
- HTML5

## Features
- Personal introduction section
- Profile image placeholder
- Hobbies and interests list
- Favorite website link
- Contact email link

## How to Run
1. Clone this repository
2. Open `index.html` in your browser

## Lessons Learned
I learned how to structure a basic HTML page using headings, paragraphs, images, lists, and links.

## Challenges Faced
One challenge was making sure the page had all the required HTML elements. I solved this by organizing the page into clear sections and checking that each required item was included.
